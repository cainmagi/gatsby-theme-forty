## Box

The `html#!<box>` element is similar to `html#!<blockquote>`, but Markdown syntax does not support it. Users could only create a new box by using the component:

<Tabs>

</box>

### Bilibili

The `jsx#!<FrameBilibili/>` for Bilibili supports both `av` and `BV` IDs. The video ID type will be resolved automatically, for example:

<Tabs>
<TabItem title='Results'>

<FrameBilibili href='av757320831'/>

which is equivalent to

<FrameBilibili href='BV1T64y1Q7i4'/>

Also, for a video with multiple pages, the argument `page` is provided for selecting a specific page.

<FrameBilibili href='BV1bW411U7sd' page='9'/>

</TabItem>
<TabItem title='Codes'>

